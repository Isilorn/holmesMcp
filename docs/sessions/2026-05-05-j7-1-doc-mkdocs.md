## Décisions prises en session

**IPs d'exemple RFC-5737** : le hook pre-push bloque les IPs privées (`192.168.x.x`) dans les commits, y compris dans la documentation. Remplacées par `203.0.113.10` (TEST-NET-3 RFC-5737), conforme et non bloquante.

**Squash obligatoire avant push** : quand un commit contient une IP bloquée et le commit suivant la corrige, le hook scanne les deux → il faut squasher en un seul commit propre avant de pousser.

**Watermark Gemini** : l'illustration source contenait un sparkle Gemini bas-droite. Supprimé par remplissage rectangle `(891,1003)→(948,1060)` avec la couleur de fond `#481a1a`. Icône et image doc régénérées depuis la source nettoyée.
