## Livraisons

### `_domain/cmd_refs.py`

Port de `jeedom-audit/resolve_cmd_refs.py`. API publique :

```python
resolve(text: str, conn) -> dict
```

Retourne :
- `resolved` : texte avec `#ID#` → `#[Objet][Équipement][Commande]#` ou `#ID_NON_RÉSOLU:X#`
- `mapping` : `{str(id): "[O][E][C]"}` pour les IDs résolus
- `unresolved` : liste d'entiers triée pour les IDs absents de la DB

Interne : `_fetch_names(ids, conn)` — jointure `cmd/eqLogic/object` en batch `%s` (PyMySQL), COALESCE objet NULL → `''`.

Adaptations vs source :
- `db_query.run(sql, params, creds)` → `db.query(conn, sql, params)` (PyMySQL local)
- Pas de cache de session global (sans état — chaque appel MCP est indépendant)
- Placeholders `%s` dynamiques (IN clause construite avec `', '.join(['%s'] * len(ids))`)
- Pas de `main()` / stdin-stdout — module bibliothèque pur
