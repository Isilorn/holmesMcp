### mcp_server.py — `_register_resources`

- 3 resources statiques globales (overview, health, logs/today)
- 2 resource templates avec paramètre URI (scenario/{id}, equipment/{id})
- Énumération D6.3 : top-50 scénarios actifs + top-50 équipements actifs (concrètes)
- Helpers : `_fetch_top_scenarios`, `_fetch_top_equipments`, `_make_scenario_fn`, `_make_equipment_fn`

**Bug corrigé** : FastMCP lève `ValueError: Mismatch between URI parameters set() and function parameters` si la fonction enregistrée pour une resource concrète (URI sans `{var}`) a des paramètres. Fix : factories sans paramètre (`_make_*_fn(id, ak) -> () -> str`).
