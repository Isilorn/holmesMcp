## Problèmes résolus

| Problème | Cause | Fix |
|---|---|---|
| `ModuleNotFoundError` (structlog, pymysql) sur pytest | Python 3.12 système sans deps | Venv dédié `/tmp/holmes-test-venv` + `[project.optional-dependencies]` dans pyproject.toml |
| 8 erreurs ruff (F401, UP035, UP041) | Imports inutilisés + stdlib dépréciés dans les tests | `ruff check --fix` + `ruff format` |
| `KeyError` dans `test_tail_returns_all_lines` | Assertion `result['error'] is None` sur clé absente | `assert 'error' not in result` |
| Auth middleware : `FastMCP` native auth incompatible | `AuthSettings` requiert `AnyHttpUrl` — trop contraignant | Middleware ASGI pur wrappant `streamable_http_app()` |

## Architecture auth retenue

```
uvicorn → BearerAuthMiddleware (ASGI pur) → mcp.streamable_http_app()
                 ↑
          TokenStore (chargé à démarrage depuis table config Jeedom)
          config::save('token_<user_id>', token, 'holmesMcp')
          JOIN user → login pour les logs downstream (scope['user'])
```
