# ADR-0013 — Sécurité opérationnelle — isolation totale des credentials

- **Date** : 2026-05-03 (esquisse) / TBD (finalisation J0)
- **Statut** : draft
- **Source brief** : §0.5 (Isolation des credentials), D11.8

## Contexte

Le repo Holmes MCP est public sur GitHub. Les sessions de développement impliquent un accès SSH à la box du PO (IP, hostname, user MySQL, tokens, apikeys). Sans garde-fous, un commit accidentel pourrait exposer ces informations. Le risque est réel : IP de box domotique, apikey Jeedom globale, password MySQL.

## Décision pressentie

**Règle absolue** : aucun credential, alias SSH, IP, hostname, user MySQL, port concret PO-spécifique, token, apikey, chemin absolu lié à l'environnement PO ne doit apparaître dans le repo Holmes MCP. Stockage exclusif sur la devbox Claude Code.

**Garde-fous techniques** (mis en place en J0) :
1. **`.gitignore` strict** : `~/.ssh/*`, `*.creds`, `*credentials*`, `.env`, `.env.local`, dumps MySQL bruts, fixtures réelles non sanitisées
2. **Pre-commit hook** : scan automatique pour patterns sensibles (regex IPv4 RFC1918, regex token format UUID/apikey, hostnames courants, patterns password) — refus du commit si détection
3. **Pre-push hook** : second scan avant push GitHub
4. **Review systématique** de chaque PR vers `main` avec checklist credentials avant merge

**Procédure de réponse rapide en cas de leak** :
1. `git filter-repo` (rewrite history) + force push
2. Rotation immédiate du credential exposé
3. ADR d'incident
4. Annonce si credential était public plus de quelques minutes

**Documentation publique** : placeholders systématiques dans tous les guides (`<your-jeedom-host>`, `<your-mysql-password>`, `<your-mcp-token>`).

## Conséquences pressenties

- Friction nulle sur les commits légitimes (les patterns légitimes ne matchent pas les regex de scan)
- Les fixtures réelles (récupérées via SSH) restent dans le workspace local Claude Code, jamais committées
- Les logs de sessions `docs/sessions/` contiennent des références neutres aux actions SSH (pas de commande brute avec paramètres sensibles)
