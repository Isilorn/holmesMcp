# Holmes MCP

> *Holmes observe, déduit, raconte. Il ne touche jamais à la scène, sans jamais exposer les secrets de votre maison.*

**Holmes MCP** est un plugin Jeedom natif open source qui expose votre box Jeedom comme serveur [MCP (Model Context Protocol)](https://modelcontextprotocol.io) en lecture seule.

Compatible avec **Claude Desktop**, **Cursor** et tout client MCP.

---

!!! warning "Documentation en cours"
    Cette documentation sera complétée au fil des jalons J0-J7. Version cible : V1.0.0.

## Démarrage rapide

1. Installez le plugin depuis le **market Jeedom**
2. Installez les dépendances Python (bouton "Dépendances")
3. Démarrez le daemon
4. Générez un token pour votre utilisateur Jeedom
5. Collez l'URL MCP + Bearer token dans votre client
